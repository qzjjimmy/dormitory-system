## 已确认模块

学生端：

- 我的宿舍：宿舍信息与室友
- 报修申请：提交维修申请
- 费用查询：水电费账单与缴费
- 公告浏览：查看系统公告
- 访客预约：提交访客来访申请
- 调宿申请：申请换宿舍/换床位
- 个人中心：个人信息管理
- AI智能问答：宿舍相关问题咨询
- 意见反馈：提交意见与建议

管理员端：

- 控制台：数据概览与统计图表
- 楼栋管理：宿舍楼 CRUD
- 房间管理：房间信息与床位配置
- 学生管理：学生信息与入住状态
- 宿管员管理：宿管人员配置
- 费用管理：水电费缴费记录
- 公告管理：系统公告发布

宿管员端：

- 工作台：待办事项与快捷入口
- 入住登记：学生入住/退宿办理
- 报修处理：维修申请审核与派单
- 访客登记：来访人员登记管理
- 卫生检查：宿舍卫生打分记录
- 晚归登记：晚归学生记录
- 物品出入：大件物品出入登记
- 消息通知：向学生发送通知

## 风格判断

B 站页面当前触发验证码，无法稳定抓取完整视频流；本项目按页面标题、简介和常见 SpringBoot + Vue 毕设演示系统风格复刻：

- 左侧深色垂直菜单
- 顶部白色工具栏和用户信息
- 蓝青色主色调
- 控制台统计卡片和图表
- 表格列表 + 搜索 + 新增/编辑表单
- 三角色菜单权限隔离
- 登录页提供角色账号入口
